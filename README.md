## API

### preference(prefs)

#### prefs

*Required*  
Type: `object`

A key-value pair object where the key is the name of the preference and the value the value.

### preference(name, value)

#### options

##### name

*Required*  
Type: `string`

The name of the preference.

##### value

*Required*  
Type: `string`

The value of the preference.
